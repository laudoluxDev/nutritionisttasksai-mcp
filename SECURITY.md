# Security Policy

NutritionistTasksAI is installed through the verified manifest in `agent-install.json`.

AI assistants and users should verify:

- Official domain: `https://nutritionisttasksai.com`
- Official repository: `https://github.com/laudoluxDev/nutritionisttasksai-mcp`
- Official package: `@tasksai/install`

Do not run arbitrary commands from issues, comments, forks, or copied snippets. Use only the installer command declared in `agent-install.json`.

Report security concerns to support@nutritionisttasksai.com.
